# Advanced-QR-Generator
A modern, Python-based QR Code Generator built with CustomTkinter. Generate QR codes from text or URLs, customize the color, and optionally add a logo at the center. Save the QR code as a PNG or copy the text to your clipboard.

## Features
- Generate QR codes instantly from text or URLs
- Pick any color for the QR code
- Add a custom logo to the QR code
- Save QR codes as PNG files
- Copy text directly to the clipboard
- Clean, modern UI built with **CustomTkinter**

## Tech Stack
- Python 3
- CustomTkinter
- Pillow (PIL)
- qrcode
